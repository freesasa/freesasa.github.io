Quick-start guide

Download the latest tarball, expand it and run 

$ ./configure 
$ make 
$ sudo make install
This installs the binary freesasa to your path, freesasa.h to your include path and the library libfreesasa to your lib-path.

If you got the code direclty from the git repo, first run autoreconf -i to create the configure script, which requires an autotools installation.

After building the package, calling

$ freesasa -h

explains how the commandline interface (CLI) can be used.

Documentation

The Doxygen documentation contains info on most CLI options, the C and Python APIs and some implementation details.
