The module provides Python bindings for the FreeSASA C Library. It works with Python 2.7 and 3.4+, on Linux, Mac OS X and Windows. The source is available as a PyPi source distribution and on GitHub.

Install the FreeSASA Python Module by

pip install freesasa

Developers can clone the library, and then build the module by the following

git clone https://github.com/freesasa/freesasa-python.git
git submodule update --init
python setyp.py build

Tests are run by

python setup.py test
